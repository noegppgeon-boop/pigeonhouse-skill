# pigeonhouse-skill

AI coding skill for [PigeonHouse](https://941pigeon.fun) — a deflationary token launchpad on Solana where every trade permanently burns PIGEON.

## What This Covers

- **Token creation** on bonding curves (Token-2022)
- **Buy/Sell** with multi-quote support (PIGEON, SOL, SKR)
- **Bonding curve math** (constant product AMM)
- **Fee structure** and automatic PIGEON burn (1.5% per trade)
- **Graduation** to Raydium CPMM with locked LP
- **Referral system** (0.5% fee share)
- **API integration** (REST endpoints + Solana Actions/Blinks)
- **On-chain data** reading (Anchor IDL decoding)
